🌱 GreenMind v3

AI-Powered Crop Disease Diagnosis Platform

🚀 Overview

GreenMind v3 is an AI-driven web application that helps farmers identify crop diseases by uploading images. The system analyzes the image using a trained machine learning model and provides:

Disease prediction

Confidence score

Observations

Recommended treatments

Suggested pesticides

This project is built as a full-stack MERN + AI application and is designed to scale into a real AgriTech platform.

🎯 Problem Statement

Farmers often:

Misidentify crop diseases

Apply incorrect pesticides

Lose yield due to delayed diagnosis

GreenMind aims to provide instant, AI-based diagnosis accessible through a simple web interface.

🔄 System Architecture

User uploads crop image

Image stored temporarily

Backend sends image to ML model

Model returns:

Disease Name

Confidence %

Backend sends structured response to frontend

UI displays:

Diagnosis

Observation

Treatment

Pesticide suggestions
